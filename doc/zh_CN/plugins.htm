1   Introduction

1.1   BlogEdit (Deprecated)

Can be used to write blogs, and communicate with a server via xml-rpc. Requires that the server supports blogapi and metawebblogapi. This plugin needs more testing.

1.2   closeother

Close others documents except current one. Close all other documents except the current one. Installation will add menu items to the File menu and to the document tab context menu.

1.3   djangoproj

Django<http://www.djangoproject.com/> support. Django is a web framework.

1.4   docbook

Can be used for writing docbook documents.

1.5   dosprompt (Windows platform)

Open DOS window in bottom pane.

1.6   fortran

Add fortran language syntax highlight support.

1.7   jQuery

jQuery support.

1.8   outline (Under development)

Like leo<http://webpages.charter.net/edreamleo/front.html>.

1.9   pairprog

You can use it on collaborative programming.

1.10   pymusic

Play mp3 music.

1.11   tts_plugin (Windows platform)

Read current document or selected text via TTS<http://en.wikipedia.org/wiki/Speech_synthesis>.

1.12   rssreader (Deprecated)

A test rss reader plugin.

1.13   ruby

Ruby support.

1.14   ruler

Add ruler for document.

1.15   showerror

Add some menu items related to error.txt and debug.txt of UliPad.

1.16   spellcheck

Spell check for current document.

1.17   wizard

Wizard framework.

1.18   mako_plugin

mako<http://www.makotemplates.org/> is a python template system. This plugin can provide mako project name, and also support .mko file suffix. When you open a .mko file, this plugin also can support simple syntax highlight. And also support some input assistant, for example: key words prompt, etc.

1.19   batchfilenames_plugin

This plugin can be used to rename batch filenames. It supports regular find and replace, simple template creation.

1.20   svn_plugin

Subversion version control support.

1.21   emacskey

Emacs keybinding.

1.22   winpdb (Discarded)

It has been merged into UliPad. So if you found this plugin displayed in plugin manager dialog, maybe you are using the old version or winpdb plugin folder is still exists in plugins folder. You should upgrade to the latest version, and check whether the plugins/winpdb is still exists, if it's existed, just remove it.
